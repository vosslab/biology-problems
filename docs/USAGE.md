# USAGE.md

## Quick start
- `source source_me.sh`
- `python3 problems/biochemistry-problems/Henderson-Hasselbalch.py -m -d 5`
- Review the generated `bbq-*.txt` output in the repo root.

## Common patterns
- Most generators support `--help`:
  - `python3 problems/inheritance-problems/<script>.py --help`
- Inspect YAML inputs:
  - `python3 data/check_yaml.py data/genetic_disorders.yml`
